# Мой проект 

## Цель моего проекта
Цель моего проекта это показать что глобальное потепление это важная проблема и хочет помочь людям ето предотвротить 

## Функции
1. start - стартует бота и обьясняет все функции
2. image - Присылает мемчик на тему ГП
3. explain - Обьясняет что такое ГП
4. tips - Дает советы как боротся с ГП
5. facts - Дает факты о ГП
6. bye - Укончает бот

## Как использовать мой проект
В VS code добавте файл .env и все файлы из папки М8У1 и в файл .env добавте переменную TG_API_TOKEN и в ее добавте свой токен
